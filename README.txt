Pour compiler le programme il faut faire les instructions suivantes :
-Ouvrir le terminal.
-Se rendre dans le dossier du ficher à compiler.
-ensuite écrite la commande suivante : g++ nomFichier.cpp -o nomFichier.exe
une fois que la commande validée alors la compilation se validera ou sinon nous allons voir les erreurs s'il y en a. 

Pour exécuter le programme il faur faire les instructions suivantes : 
-Ouvrir le terminal.
-Se rendre dans le dossier du ficher à compiler.
-ensuite écrite la commande suivante : ./nomFichier.exe
-une fois la commande validée alors on verra apparaitre les résultats de notre code.
